# lab.js_translate
Python scripts for automatic lab.js JS file translation.

These three scripts use a LLM API to translate JS scripts written in lab.js to the target language. Only frontfacing text will be translated. No CSS, HTLM, JS script will be changed

Please read "instruction.txt" for explanation on how to use the scripts. 

## Installation and preperation
1. Clone the repository
2.	Install Python
3.	Install dependencies
    ``` pip install -r requirements.txt ```
4.	Set up your LLM API key
   
	•	Follow the steps explained by DeepL or OpenAI
5. Read instruction.md
